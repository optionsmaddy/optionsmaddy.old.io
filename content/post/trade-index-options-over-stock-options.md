# Trade Index Options Over Stock Options
- Index have to be closed if it drops 20%.
- For intraday one can set lower limits as 20%
- Low Volatility. 
- Safer : Nifty Index
- more risky : Bank Nifty 
- Biggest advantage in Index : No physical delivery.
